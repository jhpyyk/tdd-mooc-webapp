# Running tests 

## Frontend
Run the tests in the frontend directory
```
cd frontend
```

### Unit tests
npm run test

### E2E test
```
./run_e2e_tests.sh
```


## Backend
Run the tests in the backend-go directory
```
cd backend-go
```

### Unit tests
go test ./...

### Integration tests  
```
./run_integration_tests.sh
```
